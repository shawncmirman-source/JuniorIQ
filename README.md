# JuniorIQ
PWA suite of learning tools for pre-k through 1st grade students.

![JuniorIQ](https://img.shields.io/badge/Status-Active-success)
![Version](https://img.shields.io/badge/version-0.0.0-blue)

**JuniorIQ** is a premium, interactive educational platform designed for early learners (Pre-K to Grade 1). It features a "Primary Playground" aesthetic with vibrant colors, smooth animations, and a focus on speed and accuracy across multiple subjects.

## 🚀 Features

### 🧠 Core Subjects
- **Math Games**: Subitize Safari (ten-frame dot recognition), Big & Small (comparative quantity).
- **Letter Learner**: Letter Pop (uppercase/lowercase matching), First Sounds (phonics matching).
- **Reading Library**: Sight Word Spark (speed-reading), Rhyme Time (auditory/visual rhyming).
- **Learning Patterns**: The Missing Link (pattern completion), Shape Shifter (visual discrimination).
- **Explore Science**: Living or Not (classification), Lifecycle Shuffle (sequential ordering).

### 🏆 The Ultimate Challenge (Quiz All)
- A 15-question mixed-subject quiz.
- Real-time results featuring a **Radar Chart** visualizing overall proficiency across all skill areas.
- Detailed tabular breakdown of accuracy and speed.

### ⚙️ Mechanics
- **State Management**: Real-time tracking of answers, accuracy, and average response time.
- **Persistence**: Local storage integration for session-based progress.
- **Scoring**: Advanced algorithm converting accuracy and speed into a 0–100 proficiency score.

## 📦 Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Custom Variables & Keyframes)
- **Framework/Bundler**: [Vite](https://vitejs.dev/)
- **Data Visualization**: [Chart.js](https://www.chartjs.org/)
- **Typography**: [Fredoka (Google Fonts)](https://fonts.google.com/specimen/Fredoka)
- **Sound**: Web Audio API (Live-generated Oscillators)

## 🏃 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shawncmirman-source/JuniorIQ.git
   ```
2. Navigate into the directory:
   ```bash
   cd JuniorIQ
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local Vite development server:
```bash
npm run dev
```

### Production Build
Create a production-ready build:
```bash
npm run build
```
To preview the build locally:
```bash
npm run preview
```
